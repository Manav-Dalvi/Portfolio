# Manav Dalvi — Personal Portfolio

Personal portfolio website showcasing my skills, projects, work experience, education, and certifications as a Data Analyst.

<a href="https://manav-dalvi.github.io/Portfolio/" target="_blank">**Visit Now** 🚀</a>

## Tech Stack

Plain HTML, CSS, and JavaScript — no build step, deployed on GitHub Pages.
Vendor libraries (jQuery, Typed.js, Vanilla Tilt, ScrollReveal, Isotope, EmailJS, particles.js) are loaded from CDNs.

## Project Structure

```
Portfolio/
├── index.html              # Main single-page portfolio
├── 404.html                # Custom 404 page
├── assets/
│   ├── css/                # Modular stylesheets (base + one file per section)
│   ├── js/
│   │   ├── vendor/         # Third-party libraries served locally
│   │   ├── main.js         # Navigation, typing effect, scroll animations
│   │   ├── render.js       # Data-driven section rendering
│   │   ├── contact.js      # Contact form (EmailJS + mailto fallback)
│   │   ├── particles-config.js
│   │   └── 404.js
│   ├── data/               # Site content as JSON (skills, projects,
│   │                       #   education, experience, certifications)
│   └── images/
├── projects/               # Standalone all-projects page with filters
└── experience/             # Standalone experience page
```

## Updating Content

All section content lives in `assets/data/*.json` — edit those files to update
skills, projects, education, experience, or certifications. No HTML changes needed.
